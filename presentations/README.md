# Conference Presentations

This directory contains presentations given by AI Vision Forum members at various conferences and events.

## Structure

Each presentation is organized in its own directory with the following naming convention:
```
conference-name-location-year/
```

Each presentation directory contains:
- `README.md` - Conference details, speakers, and presentation information
- Presentation files (PDFs, slides, etc.)
- Supporting materials
